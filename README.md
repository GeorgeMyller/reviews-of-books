Reviews of Books
This repository contains the code for a web application that analyzes and visualizes book reviews. The app uses Streamlit, Pandas, and Plotly to provide an interactive experience for exploring book data.

Features
Data visualization: The app displays various charts and graphs to help users understand the distribution of book prices, publication years, and other relevant information.
Interactive filters: Users can filter the data based on price range and other criteria to focus on specific areas of interest.
User-friendly interface: The app is designed to be easy to use and navigate, even for users with no prior experience with data analysis.

Getting Started
To run the app, follow these steps:

Clone this repository to your local machine.
Install the required Python libraries:
- streamlit
- pandas
- plotly

Run the app using the following command:
streamlit run Dashboard.py

Data Sources
The app uses two datasets:
customerreviews.csv: This dataset contains customer reviews of books.
Top_100_Trending_Books.csv: This dataset contains information about the top 100 trending books.

# Reviews of Books
Este aplicativo analisa e visualiza resenhas de livros. Ele usa Streamlit, Pandas e Plotly para fornecer uma experiência interativa para explorar dados de livros.

Recursos
Visualização de dados: O aplicativo exibe vários gráficos e tabelas para ajudar os usuários a entender a distribuição de preços de livros, anos de publicação e outras informações relevantes.
Filtros interativos: Os usuários podem filtrar os dados com base na faixa de preço e outros critérios para se concentrar em áreas específicas de interesse.
Interface amigável: O aplicativo é projetado para ser fácil de usar e navegar, mesmo para usuários sem experiência anterior com análise de dados.

Começando
Para executar o aplicativo, siga estas etapas:

Clone este repositório para sua máquina local.
Instale as bibliotecas Python necessárias:
- streamlit
- pandas 
- plotly

Execute o aplicativo usando o seguinte comando:
streamlit run Dashboard.py

Fontes de dados
O aplicativo usa dois conjuntos de dados:
customerreviews.csv: Este conjunto de dados contém resenhas de livros de clientes.
Top_100_Trending_Books.csv: Este conjunto de dados contém informações sobre os 100 livros mais populares.
